# cesiumVueEmtry
##项目结构
```
views 存放级别最高的组件 
CesiumContainer 存放map3d  主页面的组件可以并集存放
unit 存放非地图相关的工具类
config 存放地图的配置文件
component 存放组件  
    ---map3d 地图初始化 存放地图相关的基本配置
assets css icon images 静态资源库
```
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
